RSTP
Bridge settings:
  • RSTP State - STP/RSTP operating mode.
  • Bridge Priority - bridge (switch) priority set in increments of 4096 (0-61440). It is required to determine Root Bridge (of the root switch).
  • Forward BPDU - forwarding of BPDU packets when the STP/RSTP protocol is disabled
  • TX Hold Count - limit the maximum number of BPDU packets sent per second (1-10).
  • Bridge Max Age - time of waiting for the bridge in seconds, after which it sends a message about rebuilding of the network (6-40 seconds).
  • Bridge Hello Time - the period of sending BPDU packets in seconds (1-2 seconds).
  • Forward Delay Time - the delay of transition of port states from the Listening and Learning state to the Forwarding state (in seconds) (4-30).

    Port settings:
  • RSTP State - including the port in the STP / RSTP protocol.
  • Port Priority - port priority (0-240) changes in steps of 16.
  • Cost - the cost of the port's path (1-200000000).
  • Auto cost -automatically select the cost of the port path. If the tick is not set, the cost of the path is taken from the Cost field.
  • Edge - manual or automatic detection of the edge port.
  • PTP - manual or automatic detection of a point-to-point connection.
