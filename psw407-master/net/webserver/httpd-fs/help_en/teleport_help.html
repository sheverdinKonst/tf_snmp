Teleport

Devices List - the list of remote devices.
  • Info - information about the device
  • Edit - Edit an entry
  • Delete - delete an entry

    Add New Remote Device - add a new entry
  • Name - description of the remote device
  • Type - type of remote device
  • IP Address - IP address of the remote device

    Inputs - setting of the input broadcast
  • Active - activity. If checked, the input state is broadcast to the remote device
  • Remote Device - remote device
  • Remote Port - number of remote device output
  • Inverse - invert the input state
  • Current State - current state of the input

    Events - setting up the broadcast of events
  • Active - activity. If this box is checked, the event is broadcast to the remote device output.
  • Remote Device - remote device
  • Remote Port - number of remote device output
  • Inverse - inversion
  • Current State - current state of the event
