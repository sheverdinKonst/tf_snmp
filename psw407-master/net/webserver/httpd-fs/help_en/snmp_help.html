SNMP Settings

SNMP settings
  • State
  • Server IP address - IP address of the server to which SNMP Traps are sent.
  • Version - SNMP protocol version. The switch supports SNMP v1 and SNMP v3.
  • Read Community - read-only community, the string used for authentication in SNMP v1.
  • Write Community - community for recording, the string used for authentication in SNMP v1.
  • Engine ID - identifier in SNMP v3.
  • User Name - user name in SNMP v3.
  • Security Level - security level in SNMP v3:
    • NoAuth,NoPriv - operation without authentication by password (only by User Name) and without encryption.
    • Auth,NoPriv - operation with Auth Password authentication (HMAC-MD5-96 method), but without encryption.
    • Auth,Priv - operation with Auth Password authentication (HMAC-MD5-96 method), and with AES-128 encryption.
  • Auth Password - password for authentication in SNMP v3.
  • Auth Password - key for the encryption method in SNMP v3.

    SNMP Traps will be sent only when events occur that are specified in the Event List tab.
