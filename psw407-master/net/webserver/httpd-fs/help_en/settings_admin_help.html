User Accounts Settings

This section contains the settings of user accounts.
All users can be divided into 2 groups:
  • Admin - users with administrator rights have unlimited authorities.
  • User - restricted users. They do not have access to the settings, but they can reboot the cameras, see statistics, use VCT and Ping utilities.

    By default, the switch has one administrator account with an unregistered login and password, so full access to the settings is possible without authentication. To restrict access you must specify a login and password for the user with Admin rights. And, if necessary, you should create one or more users with User rights.
    • User Name - Current user name
    • New User Name - New user name
    • New Password - New Password
    • Password Confirm - Password confirmation
    • Access Right - access rights: Admin or User.

    Note 1: These fields are mandatory. The maximum length is 20 characters. Input language is English.

    Note 2: The fields are case sensitive.
    Note 3: Up to 4 users are supported.
