Port Mirroring

Port mirroring feature allows you to copy traffic that is intended for a particular port to another port. This function is used to troubleshoot the network.

State - tate of the function
Target Port - receiving port, all data from the source ports will be mirrored to this port.
Source Ports - data sourcing ports
Options of source ports operation:
N - the port does not participate in mirroring
T - all outgoing data from the port are mirrored port
R - all incoming data to the port are mirrored
B - both outgoing and incoming data are mirrored
