Inputs/Outputs
The switches have digital inputs (dry contacts). The switches have 2 digital inputs. Also, when connecting the expansion board, you can use additional inputs.

Inputs setting: :
  • State - enables input operation. An alarm message will be sent, if State is in the Enable state, and Current State matches Alarm State and a tick is set in Inputs/Outputs in the Events List tab.
  • Alarm State - the state of the input, which is considered to be alarm (Open - open state, Short - closed state).
  • Current State - current state of the input.

    Outputs settings:
  • Inital State - initial state of the output. After the power is turned on, the output will be in this state. If the output is configured as Logic, then it is in the state defined in the Logic table

    Logic
    It is possible to configure the simplest algorithms for controlling the output. For each input, you can set an event by which the input will go to the selected state:
  • Open - relay outputs are open
  • Short - relay outputs are closed
  • Impulse - relay outputs change their state to inverse for 10 seconds, then switch back to the initial state.
    Event field specifies one or more events that will trigger the port. The unification of events occurs according to the rule of logical "OR", which means the output will trigger when at least one condition occurs.
    If none of the conditions is satisfied, the output is in the inverse state.
