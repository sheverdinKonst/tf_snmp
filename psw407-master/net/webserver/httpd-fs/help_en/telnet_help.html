Telnet

Telnet – protocol used for remote management of network equipment based on port 23 TCP protocol.
State – state of the protocol. By default, Telnet is enabled. If necessary, it may be disabled.
For authentication, the same login/password are used as for accessing the WEB interface.

TFTP Settings – configures the TFTP protocol. The switch can be a TFTP client to update the firmware from the TFTP server.
State - enabling TFTP protocol
Port - UDP port for TFTP protocol
