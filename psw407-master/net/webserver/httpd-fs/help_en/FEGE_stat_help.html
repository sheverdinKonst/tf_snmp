Port Status

This tab displays information about ports and their operating modes
  • State - Port state.
  • Link - Link status.
  • Speed/Duplex - Speed/Duplex
  • Flow Control - State of flow control.
  • PoE - Power over Ethernet.
  • SFP - Detailed information about the connected SFP module(DDM, manufactor, type, etc).
  • Port Summary - Detailed information about the interface.
