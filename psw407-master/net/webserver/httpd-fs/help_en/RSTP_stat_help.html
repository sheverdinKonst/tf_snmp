STP/RSTP protocol statistics

Bridge status:
  • STP/RSTP state - protocol activity on the switch.
  • Brige Root status - the switch is the root switch.
  • Protocol - the protocol used.
  • Root bridge MAC - address of the root switch.
  • Root bridge priorityt - priority of the root switch.
  • Root port - the port directed towards the root switch.
  • Path cost to root - the cost of the path to the root switch.
  • Designated bridge MAC - address of the designated switch.
  • Designated bridge priority - priority of the designated switch.
  • Bridge max age - time of waiting for the bridge in seconds, after which it sends a message about rebuilding of the network.
  • Bridge Hello Time - the period of sending BPDU packets in seconds.
  • Forward Delay Time - the delay of transition of port states from the Listening and Learning state to the Forwarding state (in seconds).
  • Time topology change - the time elapsed since the last topology change in seconds.
  • Topology change count - the number of topology changes.

    Port status
  • Port - switch port.
  • State - STP/RSTP protocol activity on the port.
  • Baud rate - speed of the port.
  • Link - link status on the port.
  • Duplex - Duplex/Half Duplex.
  • Port state - port state. The port can be in the following states:
    STPRSTP
    ForwardingForwarding
    DisabledDiscarding
    ListeringDiscarding
    LearningLearning
    BlockingDiscarding
  • Port role - port role. The following port roles are possible:
    1. Disabled
    2. Root
    3. Designated
    4. Alternate
    5. Backup
  • Port priority - priority of the port.
  • Patch cost - cost of path through the port.
  • P2P - point-to-point connection.
  • Edge - edge port.
  • Forward transitions - the number of transitions to the Forwarding state.
