Main

The main page contains summary information:
  • Device name - device name
  • Device location - device location
  • Contact information - contacts of the servicing organization
  • Serial number - the serial number of the device
  • Firmware version - firmware version
  • Bootloader version - bootloader version
  • MAC - MAC address of the device
  • IP - network address of the device
  • Mask - Subnet Mask
  • Gateway - network address of the gateway
  • DNS - network address of the DNS server
  • DHCP - DHCP mode: static address or DHCP client.
  • System Up Time - switch operation time
  • Time, Date - the time and date received using the SNTP protocol.
  • UPS - presence of an uninterruptible power supply.
  • Battery voltage - voltage on the UPS battery (only for devices with UPS support)
  • Power source - power source: Battery or mains voltage (only for devices with UPS support)
  • Estimated battery time - estimated operating time of the device after switching to battery power (only for devices with UPS support)
  • STP/RSTP - backup protocols: STP or RSTP
  • VLAN - list of active VLANs
  • QoS - activation of Quality of Service mechanism
  • IGMP Snooping - support of multicast group management mechanism
  • Comfort Start - safe start of the cameras
  • Auto Restart - automatic restart of hanging cameras
  • Syslog - sending event messages via the Syslog protocol
  • SNTP - receiving the system time via SNTP
  • SNMP - management and monitoring via SNMP protocol
  • SMTP - sending event messages via SMTP
