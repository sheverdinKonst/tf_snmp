Virtual Cable Tester

 The cable tester is used to monitor the integrity of the UTP/FTP cable on Fast Ethernet ports.

Calibrate

 Calibration is recommended to improve the accuracy of the cable tester (± 2 m).

For this:
  1. 1. Disconnect the UTP/FTP cable from the IP camera (while the other end of the cable must remain connected to the switch).
  2. 2. Enter the actual length of the cable (in meters).
  3. 3. Press the Set button.
  4. 4. Connect the UTP/FTP cable back to the IP camera.

Possible mistakes:
  • Incorrect value - an incorrect value was entered.
  • Еrror calibration - calibration error.

Diagnostic
The table displays the test results:
  • Short - Short circuit between pairs
  • Open - A break or cable is not connected
  • Good - No damage


    • Note 1. In the event of a break or short circuit, the distance to the fault is indicated.
