RS-485

RS485 Settings - RS485 settings. If power metering device is connected using the RS485 interface, the following parameters must be set:
  • Baudrate - speed of the port (bits per second)
  • Parity - parity check: Disable, Even, Odd
  • Data Bits - number of data bits
  • Stop Bits - number of stop bits.

    Energy Meter Settings - configuring the energy meter.
  • Model - energy meter model
  • Identification - device ID
  • Password - access password
    If the device ID and password are not specified, the interaction with the energy meter will occur without a password (if allowed by the settings of the energy meter).

    Energy Meter - readings of the meter.
  • Model - model
  • State - state of connection, result of reading data
  • Total indications - total power consumption indications
  • T1, T2, T3, T4 - indications by tariff
