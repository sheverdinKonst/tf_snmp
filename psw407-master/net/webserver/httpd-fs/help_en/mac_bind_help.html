MAC Address Filtering
Ports Settings - Port State. To enable filtering by MAC addresses, it is necessary to transfer to the Secure state. For normal operation, it is necessary to transfer to Normal state.

There are several filtering options:
  • Secure: MAC filtration - only MAC address filtering; only allowed addresses pass through, the rest are blocked.
  • Secure: Port shutdown - MAC address filtering, if unallowed MAC address is detected on the port, the port is turned off.
  • Secure: Temporary port shutdown - MAC address filtering, if an unallowed MAC address is detected on the port, the port is turned off for 5 minutes. If MAC is lost, the port is turned on; if not, the port remains off.

    Add New MAC Address - add a new allowed MAC address.

    MAC Address Table - table of added allowed MAC addresses.

    Note 1: in total, up to 50 entries of allowed MAC addresses are supported on all ports.
