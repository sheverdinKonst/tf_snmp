Network Settings

This section describes the basic network settings of the switch.
  • MAC - The physical address of the device.
  • IP - the network address of the device.
  • Mask - the subnet mask.
  • Gateway - Gateway network address.
  • DNS - network address of the DNS server
  • DHCP Mode - selecting the operating mode of the DHCP protocol:
    • Disable - DHCP disabled. The switch uses static settings.
    • Client - DHCP mode enabled. The switch receives network settings automatically.
  • Gratuitous ARP - When this option is enabled, the switch will receive and send gratuitous ARP

Note: If the gateway is not used, leave the default value: 255.255.255.255
