Teleport

Devices List - список удалённых устройств.
  • Info - информация о устройстве
  • Edit - редактирование записи
  • Delete - удаление записи

    Add New Remote Device - добавление новой записи
  • Name - описание удалённого устройства
  • Type - тип удалённого устройства
  • IP Address - IP адрес удалённого устройства

    Inputs - настройка трансляции входов
  • Active - активность. Если галочка установлена, состояние входа ретранслируется на удалённое устройство
  • Remote Device - удалённое устройство
  • Remote Port - номер выхода удалённого устройства
  • Inverse - инвертировать состояние входа
  • Current State - текущее состояние входа

    Events - настройка трансляции событий
  • Active - активность. Если галочка установлена, событие ретранслируется на выход удалённого устройства
  • Remote Device - удалённое устройство
  • Remote Port - номер выхода удалённого устройства
  • Inverse - инверсия
  • Current State - текущее состояние события
