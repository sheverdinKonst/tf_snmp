Настройки учетных записей пользователей

В данном разделе содержатся настройки учетных записей пользователей.
Всех пользователей можно разделить на 2 группы:
  • Admin - пользователи с правами администратора, имеют неограниченные полномочия.
  • User - пользователи с ограниченными правами. Не имеют доступ к настройкам, но могут перезагружать камеры, смотреть статистику, использовать утилиты VCT и Ping.

    По умолчанию, на коммутаторах создана одна учётная запись администратора с неустановленными логином и паролем, поэтому полный доступ к настройкам возможен без аутентификации, для ограничения доступа обязательно следует как минимум задать логин и пароль для пользователя с правами Admin. И, при необходимости можно создать одного или нескольких пользователей с правами User.

    Примечание 1: Данные поля являются обязательными для заполнения. Максимальная длина - 32 символа. Язык ввода - английский.

    Примечание 2: Поля регистрозависимые. Примечание 3: Поддерживается до 4 пользователей.
