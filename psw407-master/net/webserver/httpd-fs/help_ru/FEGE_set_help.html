Настройки портов
  • Состояние - Состояние порта.
  • Скорость/Дуплекс
  • Управление потоком - Управление потоком передачи данных. Flow Control активен при Speed/Duplex=Auto
  • PoE - Power over Ethernet (питание поверх Ethernet). Существует несколько вариантов подачи питания:
    • Disabled - PoE выключено на этом порту.
    • Auto - Основной режим, питание передаётся по варианту А или B, или А и B по запросу запитываемого оборудования.
    • UltraPoE - Передача питания одновременно по А и В. Адаптация для подключения купольных камер Beward.
    • Only A - Передача питания осуществляется только по варианту А.
    • Only В - Передача питания осуществляется только по варианту В.
    • Passive B - Передача питания осуществляется по варианту А в протокольном режиме и по варианту B в беспротокольном режиме. Режим для работы совместно с термокожухами TFortis TH-02 и TFortis TH-03.
    • Forced A+B - Передача питания осуществляется по варианту А и B в беспротокольном режиме. Подключение non-PoE устройств запрещено.
  • Режим SFP линка - режим работы линка SFP порта.
    • Forced - форсированнное поднятие линка при обнаружении сигнала по оптике
    • Auto - автоматический режим


Примечание 1 (для устройства PSW-2G+): Для варианта PoE B есть возможность устновить режим Passive PoE. Данный режим необходим для подключения термокожухов TH-02 и TH-03. Следует иметь в виду, что в этом режиме напряжение (55 В) подаётся на порт Fast Ethernet, поэтому подключение устройств, не поддерживающих PoE(например компьютер) может привести к выходу их из строя. При необходимости локального подключения к портам Fast Ethernet коммутатора PSW-2G+ необходимо использовать 4-х проводный (2-х парный) патч-корд или осуществлять подключение через порты Gigabit Ethernet
