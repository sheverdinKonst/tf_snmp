Входы/Выходы
Коммутаторы имеют цифровые входы(сухие контакты). В коммутаторах есть 2 цифровых входа(Sensors), кроме того при подключении платы расширеня можно использовать дополнительные входы (Inputs).

Настройка входов:
  • State - разрешение работы входа. Аварийное сообщение будет отправлено, если State установлено в Enable, и произошло совпадение Alarm State и Current State, а также установлена галочка Inputs/Outputs во вкладке Events List.
  • Alarm State - состояние входа, которое считается аварийным( Open - разомкнутое состояние, Short - замкнутое состояние).
  • Current State - текущее состояние входа.

    Настройка выходов:
  • Inital State - начальное состояние выхода. После включения питания выход будет находиться в этом состоянии. Если выход настроен как Logic, то он находится в состоянии, определяемом в таблице Logic(Логика)

    Логика
    Существует возможность настроить простейшие алгоритмы для управления выходом. Для каждого входа можно установить событие, по которому вход перейдёт в выбранное состояние:
  • Open - выходы реле разомкнуты
  • Short - выходы реле замкнуты
  • Impulse - выходы реле изменяют своё состояние на инверсное на 10 секунд, затем переключаются обратно в начальное состояние.
    В поле Event указывается одно или несколько событий, по которым будет срабатывать порт. Объединение событий происходит по правилу логического "ИЛИ", т.е. выход будет срабатывать при наступлении хотя бы одного условия.
    Если ни одно условие не выполняется, то выход находится в состоянии, инверсном установленному.
