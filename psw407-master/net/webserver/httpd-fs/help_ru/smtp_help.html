Настройки SMTP
  • Состояние
  • IP адрес сервера - IP адрес почтового сервера.
  • Доменое имя сервера
  • Порт - номер TCP порта, через который происходит отправка писем (0-65534). По умолчанию 25.
  • Адрес отправителя - почтовый адрес отправителя. (From)
  • Адрес получателя 1-3 - почтовый адрес получателя.(To). Может быть до 3 адресов.
  • Тема письма - тема письма.
  • Логин, Пароль - логин и пароль, если почтовый сервер требует процедуру аутентификации. Если поля пустые, то будет использоваться работа без аутентификаци
    Коммутатор поддерживает методы аутентификации: AUTH PLAIN и AUTH LOGIN.

    Send test e-mail (послать тестовое e-mail сообщение)
  • Тема письма - заголовок письма(до 16 символов)
  • Сообщение - текст письма (до 64 символов)
    Язык ввода - английский

    SMTP сообщения будут посылаться только при наступлении тех событий, которые указанны во вкладке Список событий

    По умолчанию, если указан IP адрес сервера, сообщения направляются на этот адрес. Для использования доменного имени, установите IP адрес сервера в 0.0.0.0 и укажите доменное имя сервера.
