Фильтрация по MAC адресам
Ports Settings - Состояние порта. Для включения фильтрации по МАС адресам необходимо перевести в состояние Secure. Для обычного функционирования - в состояние Normal.
Существуют несколько вариантов фильтрации:
  • Secure: MAC filtration - только фильтрация MAC адресов, пропускаются только разрешённые, остальные блокируются.
  • Secure: Port shutdown - фильтрация MAC адресов, если на порту обнаружен неразрешённый MAC адрес, порт выключается.
  • Secure: Temporary port shutdown - фильтрация MAC адресов, если на порту обнаружен неразрешённый MAC адрес, порт выключается на 5 минут. Если МАС пропал, то порт включается, если нет, то порт остаётся в выключенном состоянии.

    Add New MAC Address - добавить новый разрешённый MAC адрес.

    MAC Address Table - таблица добавленных разрешённых MAC адресов.

    Примечание 1: суммарно на всех портах поддерживается до 50 записей разрешённых MAC адресов.
