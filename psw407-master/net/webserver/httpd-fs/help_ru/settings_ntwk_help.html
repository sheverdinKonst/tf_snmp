Сетевые настройки

В данном разделе указываются основные сетевые настройки коммутатора
  • MAC - Физический адрес устройства.
  • IP - Сетевой адрес устройства.
  • Маска - Маска подсети
  • Шлюз - Сетевой адрес шлюза
  • DNS - Сетевой адрес DNS-сервера
  • Режим DHCP - выбор режима работы с протоколом DHCP
    • Disable - DHCP отключен. Коммутатор использует статические настройки.
    • Client - включене режима DHCP клиента. Коммутатор получает сетевые настройки автоматически.
  • Самообращённые ARP - При включении этой опции коммутатор будет принимать и рассылать самообращённые ARP (Gratuitous ARP)

Примечание: Если шлюз не используется, то оставьте значение по умолчанию: 255.255.255.255
