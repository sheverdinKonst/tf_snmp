RS-485

RS485 Settings - настройки RS485. Если иcпользуется подключение прибора учёта электроэнергии по интерфейсу RS485, то необходимо настроить следующие параметры:
  • Baudrate - скорость порта(бит в секунду)
  • Parity - проверка на чётность: Disable - отключена; Even - Чет.;Odd - Нечет.
  • Data Bits - число бит данных
  • Stop Bits - число стоповых битов

    Energy Meter Settings - настройка прибора учёта.
  • Model - модель прибора учёта
  • Identification - идентификатор устройства
  • Password - пароль доступа
    Если идентификатор устройства и пароль доступа не заданы, то взаимодействие со счётчиком будет происходить в безпарольном режиме(если разрешено в настройках самого счётчика).

    Energy Meter - показания прибора учёта.
  • Model - модель
  • State - состояние подключения, результат считывания данных
  • Total indications - суммарные показания расхода электроэнергии
  • T1, T2, T3, T4 - показания по тарифу
