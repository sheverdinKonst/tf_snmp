Настройки SNMP
  • Состояние - состояние.
  • Режим - режим работы: только чтение, только запись, чтение и запись.
  • Сервер для приема трапов - IP адрес сервера, на который отправляются SNMP Traps. Если поле оставить пустым (0.0.0.0), то трапы отправляться не будут
  • Версия - версия протокола SNMP. Коммутатор поддерживает SNMP v1 и v3.
  • Сообщество чтения - сообщество только для чтения, строка использующаяся для аутентификации в SNMP v1.
  • Сообщество записи - сообщество для записи, строка использующаяся для аутентификации в SNMP v1.
  • Engine ID - уникальный идентификатор в SNMP v3.
  • Имя пользователя - имя пользователя в SNMP v3.
  • Уровень безопасности - уровень безопасности в SNMP v3:
    • NoAuth,NoPriv - работа без авторизации по паролю(только по User Name) и без шифрования.
    • Auth,NoPriv - работа с использованием авторизации по паролю Auth Password (метод HMAC-MD5-96),но без шифрования.
    • Auth,Priv - работа с использованием авторизации по паролю Auth Password (метод HMAC-MD5-96),и с шифрованием AES-128.
  • Auth Password - пароль для авторизации в SNMP v3.
  • Auth Password - ключ для метода шифрования в SNMP v3.

    SNMP Traps будут посылаться только при наступлении тех событий, которые указанны во вкладке Event List и если указан IP адрес сервера.
