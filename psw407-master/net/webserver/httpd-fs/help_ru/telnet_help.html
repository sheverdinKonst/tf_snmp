Telnet

Telnet - протокол для реализации удалённого управления сетевым оборудованием, основанный на протоколе TCP порт 23.
Состояние - состояние протокола. По-умолчанию Telnet включен, при необходимости его можно выключить. Опция Эхо - при включении опции "Эхо", введёные символы "дублируются" в терминале.
Для аутентификации используются те же логин/пароль, что и для доступа к WEB-интерфейсу.

Настройки TFTP
Коммутатор может быть TFTP клиентом для обновления прошивки с TFTP сервера.
Состояние - включение протокола TFTP
Порт - UDP порт для протокола TFTP
